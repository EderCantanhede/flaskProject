# flaskProject

Este repositório faz parte de um conjunto de projetos de treinamento e prática de codificação oferecidos pelo [freeCodeCamp](https://www.freecodecamp.org/). O objetivo é desenvolver habilidades práticas em desenvolvimento web, utilizando **Flask** como o framework principal. 

O primeiro projeto é a criação de uma página de formulário de pesquisa, um requisito para obter a certificação na plataforma.

## Objetivo do Projeto

A página de formulário de pesquisa é projetada para:
- Permitir aos usuários preencher informações pessoais básicas.
- Coletar dados de forma segura e eficiente.
- Enviar os dados para um servidor Flask para processamento ou armazenamento.

Este projeto é uma ótima introdução ao desenvolvimento de formulários e interação com servidores backend usando Flask.

## Tecnologias Utilizadas

- **Flask**: Framework de desenvolvimento web em Python.
- **HTML/CSS**: Estrutura e estilização da página de formulário.
- **JavaScript** (opcional): Para validação de formulário e interatividade adicional.

## Estrutura do Repositório

- `/static`: Arquivos CSS e JavaScript.
- `/templates`: Arquivos HTML para o frontend.
- `app.py`: Arquivo principal do Flask que gerencia as rotas e a lógica do backend.
- `README.md`: Documentação do projeto.

## Como Executar

1. Clone este repositório:
    ```bash
    git clone https://github.com/seu-usuario/flaskProject.git
    ```
2. Instale as dependências:
    ```bash
    pip install flask
    ```
3. Inicie o servidor:
    ```bash
    python app.py
    ```
4. Acesse o formulário de pesquisa em `http://localhost:5000`.
